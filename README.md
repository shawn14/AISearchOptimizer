# AISearchOptimizer

> Track your brand's visibility across AI-powered search engines like ChatGPT, Claude, Perplexity, Google AI, and Gemini.

## What This Does

AISearchOptimizer helps marketing teams understand and improve how AI search engines talk about their brands. When someone asks ChatGPT, Claude, or Perplexity about topics in your industry, does your brand get mentioned? Where do you rank compared to competitors?

### Core Features

1. **AI Visibility Monitoring**
   - Track brand mentions across ChatGPT, Claude, Perplexity, Google AI, Gemini
   - Monitor competitor visibility
   - Track citation prominence and sentiment
   - Historical trend analysis

2. **Content Optimization**
   - AI-powered content generation with brand voice
   - Answer Engine Optimization (AEO) scoring
   - Schema.org structured data generation
   - Content gap analysis

3. **Competitive Intelligence**
   - Benchmark against competitors
   - Share of voice analysis
   - Strategy insights

4. **Automated Reporting**
   - Scheduled monitoring and alerts
   - PDF/CSV exports
   - Webhook integrations

## Technology Stack

- **Frontend**: Next.js 14+ with TypeScript, Tailwind CSS, shadcn/ui
- **Backend**: Next.js API Routes + optional FastAPI for heavy tasks
- **Database**: PostgreSQL (Supabase/Neon)
- **Cache**: Redis (Upstash)
- **AI APIs**: OpenAI, Anthropic, Perplexity, Google Gemini
- **Deployment**: Vercel
